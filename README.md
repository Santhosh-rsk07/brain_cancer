# Brain Cancer Prediction using Machine Learning

This project uses Machine Learning to predict brain cancer based on patient health data.

## Features
- Load and analyze dataset
- Train Logistic Regression model
- Predict cancer condition
- Calculate model accuracy
- Visualize patient conditions using charts

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## File
- `untitled4.py` – Main Python program for brain cancer prediction.

## How to Run
1. Install required libraries:
   ```bash
   pip install pandas matplotlib scikit-learn
2. Run the Python file:
   python untitled4.py
## Dataset Inputs
Age
Headache
MRI Score
Tumor Size
## Output
Cancer prediction result
Model accuracy
Bar chart visualization
